\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class Box$<$ T $>$}

The Box$<$\+T$>$ template represents a bounding box defined by minimum and maximum values of type T. 

The min and max members are public.

The type T must implement\+:

1) Arithmetic operator + and -\/ with the signature (T,T) and the expected return values for a numeric type.

2) Comparison operators $<$, $>$, $<$=, $>$= with the signature (T,T) and the expected return values for a numeric type.

3) Assignment operators = with the signature (T,T) and with the signature (T,float and/or double).

4) A constructor which takes a float (and/or double) for use in initializing the box.

5) A function T\+::dimensions() which returns the number of dimensions in the class (since its assumed its a vector) -- preferably, this returns a constant expression.
